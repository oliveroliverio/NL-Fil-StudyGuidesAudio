# Introduction
Welcome to this lesson on Filipino greetings.

# Vocabulary
Let's learn some basic greetings.
Magandang umaga means Good morning.
Magandang hapon means Good afternoon.
Magandang gabi means Good evening.

# Dialogue
Here is a short conversation.
Person A says: Magandang umaga, Maria.
Person B replies: Magandang umaga din, Juan. Kumusta ka?
Person A answers: Mabuti naman. Salamat.
