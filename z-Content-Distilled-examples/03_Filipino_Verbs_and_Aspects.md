# Filipino Verbs and Aspects

## 1. The Focus System
Filipino verbs are "focused" on a specific part of the sentence. The focus determines which affix to use and which noun gets the **Ang** marker.

### A. Actor Focus (Subject Focus)
The subject (Ang phrase) is the **doer** of the action.
*   **Affixes:** Mag-, Um-, Ma-, Mang-
*   **Example:** *Kumain **si** Juan ng mansanas.* (Juan ate an apple.)
    *   Focus: **Juan** (the eater).

### B. Object Focus
The subject (Ang phrase) is the **receiver** of the action.
*   **Affixes:** -in, i-, -an
*   **Example:** *Kinain ni Juan **ang** mansanas.* (The apple was eaten by Juan.)
    *   Focus: **Mansanas** (the thing eaten).

## 2. Aspects of the Verb (Tenses)
Tagalog verbs don't have "tense" (time) in the strict sense, but "aspect" (state of completion).

1.  **Infinitive (Pawatas):** The base form + affix (e.g., *Magluto*).
2.  **Completed (Perpektibo/Past):** The action is done (e.g., *Nagluto*).
3.  **Uncompleted (Imperpektibo/Present):** The action is currently happening or habitual (e.g., *Nagluluto*).
4.  **Contemplated (Kontemplatibo/Future):** The action has not started yet (e.g., *Magluluto*).

## 3. Conjugation Rules (Actor Focus)

### A. MAG- Verbs (Deliberate actions)
*   **Root:** *Luto* (Cook)
*   **Infinitive:** **Mag**luto
*   **Completed (Past):** **Nag**luto (Change M to N)
*   **Uncompleted (Present):** **Nag**luluto (Nag + First syllable reduplicated)
*   **Contemplated (Future):** **Mag**luluto (Mag + First syllable reduplicated)

| Aspect | Rule | Example (Aral - Study) | Example (Linis - Clean) |
| :--- | :--- | :--- | :--- |
| **Infinitive** | Mag + Root | Mag-aral | Maglinis |
| **Past** | Nag + Root | Nag-aral | Naglinis |
| **Present** | Nag + Rep1 + Root | Nag-aaral | Naglilinis |
| **Future** | Mag + Rep1 + Root | Mag-aaral | Maglilinis |

### B. UM- Verbs (Natural/Bodily actions)
*   **Root:** *Kain* (Eat)
*   **Infinitive:** **K**um**ain** (Infix -um- after first letter, or prefix Um- if vowel start)
*   **Completed (Past):** **K**um**ain** (Same as infinitive)
*   **Uncompleted (Present):** **K**um**akain** (-um- + Rep1)
*   **Contemplated (Future):** **K**akain (Rep1 + Root)

| Aspect | Rule | Example (Inom - Drink) | Example (Sulat - Write) |
| :--- | :--- | :--- | :--- |
| **Infinitive** | Um + Root / -um- | Uminom | Sumulat |
| **Past** | Um + Root / -um- | Uminom | Sumulat |
| **Present** | Um + Rep1 + Root | Umiinom | Sumusulat |
| **Future** | Rep1 + Root | Iinom | Susulat |

### C. MA- Verbs (Ability / State / Involuntary)
*   **Root:** *Tulog* (Sleep)
*   **Infinitive:** **Ma**tulog
*   **Completed (Past):** **Na**tulog (Change M to N)
*   **Uncompleted (Present):** **Na**tutulog (Na + Rep1)
*   **Contemplated (Future):** **Ma**tutulog (Ma + Rep1)

### D. MANG- Verbs (Professional / Collective / Multiple actions)
*   **Root:** *Isda* (Fish) -> *Mangisda* (To go fishing)
*   **Assimilation:** The prefix changes based on the root's first letter.
    *   **Mang-**: Vowels, k, g, h (*Mang-agaw, Manghingi*)
    *   **Man-**: d, l, r, s, t (*Manligaw, Manood*)
    *   **Mam-**: b, p (*Mamili, Mamigay*)

| Aspect | Rule | Example (Bili - Buy -> Mamili) |
| :--- | :--- | :--- |
| **Infinitive** | Mang/Man/Mam + Root | Mamili |
| **Past** | Nang/Nan/Nam + Root | Namili |
| **Present** | Nang + Rep1 + Root | Namimili |
| **Future** | Mang + Rep1 + Root | Mamimili |
